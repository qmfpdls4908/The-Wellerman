🌊 배 내부 물 마스킹 가이드

배가 반쯤 잠길 때 물 메쉬가 배 안까지 보이는 문제를 해결합니다

💡 원리

배 안에 보이지 않는 Quad 메쉬를 깔아둡니다. 이 메쉬가 Stencil Buffer에 "여기는 물 그리지 마"라고 표시하면, 물 셰이더가 해당 영역을 건너뜁니다.

측면도 (Side View) ~~~~~~수면~~~~~~~~~수면~~~~~~ ┌──────────────┐ ~~~~~│ 물이 보임! │~~~~~~~~~ ← 이 부분을 마스킹 │ 배 내부 │ ─────┴──────────────┴─────────
1 Layer 생성

Edit > Project Settings > Tags and Layers 로 이동합니다.

빈 슬롯에 아래 2개 Layer를 추가합니다.

슬롯이름비고
빈 슬롯WaterMask마스크 메쉬용 (새로 생성)
빈 슬롯Water이미 있으면 스킵
2 Material 생성
  1. Project 창에서 우클릭 > Create > Material
  2. 이름: WaterMask_Mat
  3. Inspector 상단의 Shader 드롭다운 클릭
  4. Custom > WaterMask 선택
📁 셰이더 파일 위치

Stylized Water Shader/Shaders/WaterMask.shader

3 마스크 메쉬 배치

3-1. Quad 생성

  1. Hierarchy에서 배(Boat) 오브젝트 우클릭
  2. 3D Object > Quad 선택
  3. 이름을 WaterMask 로 변경

3-2. Transform 설정

항목값설명
Position X0배 중앙
Position Y수면 높이에 맞춤배 기준 로컬 Y
Position Z0배 중앙
Rotation X90Quad를 수평으로 눕힘
Scale X, Z배 내부 크기에 맞춤넉넉하게, 배 밖으로 나가지 않게

3-3. Material 적용

WaterMask 오브젝트의 Mesh Renderer > Materials > Element 0 에 WaterMask_Mat 드래그

3-4. Layer 변경

오브젝트Layer 설정
WaterMask (마스크 Quad)WaterMask
Water Plane (물)Water
위에서 본 모습 (Top View) ┌──────────────────┐ ← 배 Hull │ ┌────────────┐ │ │ │ WaterMask │ │ ← Quad (배 내부에 맞춤) │ │ (Quad) │ │ │ └────────────┘ │ └──────────────────┘
4 Renderer Feature 추가

프로젝트의 URP Renderer Asset 을 선택합니다.

📍 찾는 방법

Edit > Project Settings > Graphics 에서 연결된 Asset 클릭
또는 Stylized Water Shader/URP-HighFidelity-Renderer.asset 사용

▶ Feature #1 — WaterMask_Write (마스크 기록)

Inspector 하단 Add Renderer Feature > Render Objects

설정값
NameWaterMask_Write
EventBefore Rendering Transparents

Filters 섹션

설정값
QueueOpaque
Layer MaskWaterMask 만 체크 ✓

Overrides 섹션 > ☑ Stencil 체크

설정값
Value1
Compare FunctionAlways
PassReplace
FailKeep

▶ Feature #2 — Water_StencilTest (물 마스킹)

다시 Add Renderer Feature > Render Objects

설정값
NameWater_StencilTest
EventBefore Rendering Transparents

Filters 섹션

설정값
QueueTransparent
Layer MaskWater 만 체크 ✓

Overrides 섹션 > ☑ Stencil 체크

설정값
Value1
Compare FunctionNotEqual
PassKeep
FailKeep
5 확인
  1. Play 모드 진입
  2. 배를 측면에서 바라보기
  3. 배 안에 물이 안 보이면 성공! 🎉

🔧 문제 해결

증상확인할 것
마스크가 안 됨Layer 설정 확인 (WaterMask, Water)
물이 전부 사라짐Renderer Feature의 Layer Mask가 정확한지 확인
배 밖 물도 잘림마스크 Quad가 배 밖으로 튀어나가지 않게 크기 조절
특정 각도에서 깨짐마스크 Quad의 Y 위치를 수면보다 살짝 위로 조정
