# Phase 04: Launch (Polish & Release)
## 개발 기간: 1개월 (Beta 이후)

---

## 개요
게임을 스팀 출시에 준비된 상태로 완성하는 단계입니다. 밸런스 최종 조정, 버그 수정, 최적화, 그리고 클리어 후 콘텐츠를 추가하여 재플레이성을 확보합니다.

---

## 목표
- 전체 밸런스 최종 조정
- 버그 수정 및 품질 향상
- 스팀 페이지 준비 및 빌드
- 클리어 후 콘텐츠 (해금 시스템)
- 최적화 (저사양 PC 지원)

---

## 구현 항목

### Week 1: 밸런스 및 QA

#### 밸런스 조정
- [ ] 각 역할별 업묟 분포 분석 (너무 한가하거나 바쁘지 않은지)
- [ ] 페이즈별 클리어 시간 데이터 수집 (목표: 15~20분)
- [ ] 사기 감소/회복 곡선 미세 조정
- [ ] 화약 수급량 vs 소모량 밸런스
- [ ] 고래 공격 빈도/데미지 조정 (2인 vs 4인 난이도)

#### QA 및 버그 수정
- [ ] 전체 기능 리그레션 테스트
- [ ] LAN 멀티 환경 스트레스 테스트 (4인, 장시간)
- [ ] 디스커넥션/재접속 엣지 케이스 처리
- [ ] 메모리 누수 체크 (프로파일러)
- [ ] 세이브/로드 데이터 무결성 검증

### Week 2: 최적화 및 접근성

#### 성능 최적화
- [ ] GPU Instancing 적용 (파도, 파티클)
- [ ] LOD (Level of Detail) 설정 (고래, 배)
- [ ] 오클컬링 설정 (배 낭부 가리기)
- [ ] 텍스처 아틀라스 생성 (Draw Call 감소)
- [ ] 저사양 모드 설정 (그래픽 품질 프리셋)

#### 접근성
- [ ] 치마라 흔들림 강도 조절 슬라이더 (ON/OFF → 단계별)
- [ ] 자막 설정 (사운드 효과 자막)
- [ ] 색맹 모드 (적/녹 구분 어려운 색상 변경)
- [ ] 키 리맵핑 지원 (모든 액션 키 변경 가능)
- [ ] 컨트롤러 완전 지원 (Xbox/PlayStation 패드)

### Week 3: 클리어 후 콘텐츠

#### 해금 시스템
- [ ] **고래 스킨**: 클리어 시 다른 색상/무늬의 고래 해금
  - 기본: 흰고래
  - 해금 1: 검은 고래
  - 해금 2: 황금 고래
  - 해금 3: 발광하는 깊은 바다 고래
- [ ] **배 스킨**: 클리어 횟수에 따른 배 외형 변화
- [ ] **도구 스킨**: 갈고리, 망원경 등 외형 변화

#### 도전 모드 (1회 클리어 후 해금)
- [ ] 체크포인트 없는 원모어 모드
- [ ] 시간 제한 모드 (15분 내 클리어 목표)
- [ ] 무기한 모드 (계속 오는 고래 떼)

#### 통계 및 도전과제
- [ ] 플레이 통계 (총 플레이 시간, 발사한 포탄 수 등)
- [ ] 스팀 도전과제 (업적)
  - 첫 클리어
  - 무사고 클리어 (배 한 척도 안 부서짐)
  - 4인 협동 클리어
  - 선원의 노래 난이도 클리어

### Week 4: 출시 준비

#### 스팀 페이지
- [ ] 스토어 페이지 작성 (한/영/일/중 다국어)
- [ ] 스크린샷 촬영 (4~6장)
- [ ] 트레일러 영상 편집 (30초, 60초 버전)
- [ ] 시스템 요구사항 명시
- [ ] 가격 정책 수립

#### 빌드 및 배포
- [ ] Windows 빌드 (x64)
- [ ] Mac 빌드 (Optional)
- [ ] Steamworks SDK 통합 (업적, 리더보드)
- [ ] DRM 프리 빌드 확인
- [ ] 첫날 패치 준비 (Hotfix 대응)

#### 문서화
- [ ] 유저 매뉴얼 (PDF/웹)
- [ ] 트러블슈팅 가이드
- [ ] 크레딧 제작
- [ ] 패치 노트 템플릿
