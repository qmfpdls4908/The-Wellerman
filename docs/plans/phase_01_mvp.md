# Phase 01: MVP (Minimum Viable Product)
## 개발 기간: 2개월

---

## 개요
1인칭 시점의 기본적인 게임플레이와 싱글플레이어 모드를 구현하여 핵심 게임 루프를 검증하는 단계입니다. AI 선장이 자동으로 조종하는 환경에서 1등 선원의 모든 기능을 구현하고, 기본적인 고래 AI와 전투를 완성합니다.

---

## 목표
- 1인칭 캐릭터 컨트롤러 및 상호작용 시스템 구현
- 싱글플레이어 기본 게임 루프 완성
- 기본 고래 AI (추적/돌진/공격)
- 페이즈 시스템 기초 구조 구현
- 로우폴리 아트 스타일 확립

---

## 구현 항목

### Week 1-2: 프로젝트 세팅 및 기초 시스템

#### 기술 인프라
- [ ] Unity 2022.3 LTS 프로젝트 생성
- [ ] URP (Universal Render Pipeline) 설정
- [ ] Input System 패키지 설치 및 설정
- [ ] 기본 폴리더/콜라이더 구조 설계
- [ ] Gitignore 및 버전 관리 설정

#### 1인칭 컨트롤러 기초
- [ ] FPV 치마라 리그 설정
- [ ] 기본 이동 (WASD)
- [ ] 점프/달리기
- [ ] 치마라 흔들림 시스템 기초 (ON/OFF 설정용 구조)
- [ ] 상호작용 시스템 (E 키 기반)

### Week 3-4: 배 환경 및 물리

#### 배 구조
- [ ] 해적선 기본 메쉬 모델링 (로우폴리)
- [ ] 배 낭부 공간 구성 (갑판, 선실, 포대 위치)
- [ ] Buoyancy 시스템 적용 (물에 떠다니는 물리)
- [ ] 기본 파도 물리 (단순 사인파 기반)

#### 1등 선원 역할 구현
- [ ] 수리 시스템 (부서진 선체 부분 상호작용)
- [ ] 망원경 시스템 (홀드 시 줌, 시야 제한)
- [ ] 갈고리 던지기 (보급품 회수)
- [ ] 수리 도구 UI (망치 아이콘, 진행도)

### Week 5-6: 고래 AI 기초

#### 고래 기본 동작
- [ ] 고래 로우폴리 모델링
- [ ] State Machine 기초 구조 (Idle, Track, Charge, Attack)
- [ ] 수면 위/아래 전환 애니메이션
- [ ] 돌진 패턴 구현 (직선 이동, 충돌 판정)
- [ ] 꼬리치기 공격 (배 측면 타격)

#### 전투 시스템
- [ ] 대포 기본 조작 (장전→조준→발사)
- [ ] 포탄 물리 및 충돌 판정
- [ ] 고래 체상 시스템 (HP 100%)
- [ ] 데미지 피드백 (고래 피격 반응)

### Week 7-8: AI 선장 및 게임 루프

#### AI 선장 시스템
- [ ] AI 선장 자동 조종 로직
- [ ] 자동 사기 관리 (주기적 회복)
- [ ] 전술 지시 UI (텍스트/화살표로 고래 위치 알림)
- [ ] AI 배 회피 동작

#### 페이즈 시스템 기초
- [ ] 페이즈 전환 구조 (100% → 75% → 50% → 25%)
- [ ] 페이즈별 고래 행동 변화 기초
- [ ] 첫 번째 배 파괴 이벤트 (75% 시점)

#### UI/UX 기초
- [ ] 플레이어 상태 UI (철망 없음)
- [ ] 사기 게이지 기초
- [ ] 고래 HP 바
- [ ] 상호작용 프롬프트

## 검증 기준 (완료 기준)

### 기능적 완료
- [ ] 1인칭 시점에서 배 위를 자유롭게 이동 가능
- [ ] 고래가 수면 위/아래로 이동하며 돌진 공격 수행
- [ ] 대포로 고래를 맞추고 HP가 감소함
- [ ] 선체 파손 시 수리 가능
- [ ] AI 선장이 자동으로 배를 움직이고 사기를 관리
- [ ] 고래 HP 75% 시 첫 배 파괴 이벤트 발생
- [ ] 15분 내외로 한 세션 완료 가능

### 품질 기준
- [ ] 60fps 이상 유지 (테스트 PC 기준)
- [ ] 치마라 흔들림 ON/OFF 전환 정상 작동
- [ ] 모든 상호작용 E 키로 작동
- [ ] 고래 공격이 명확한 사전 애니메이션으로 예고됨

## 리스크 및 고려사항

### 기술적 리스크
- **물리 시뮬레이션**: Buoyancy 툴킷과 Unity 물리의 충돌 가능성

- **1인칭 멀미**: 과도한 흔들림
  - 대응: Week 2부터 흔들림 ON/OFF 옵션 구현

### 일정 리스크
- **고래 AI 복잡도**: 기본 패털도 생각보다 시간 소요
  - 대응: 돌진/추적 2가지만 MVP에 포함, 나머지는 이후 단계

### 의존성
- Input System 패키지 설치 필요
- URP 설정 완료 필요 (셰이더 작업 전)
